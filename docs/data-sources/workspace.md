# ory_workspace (Data Source)

Fetches information about an Ory workspace.

This data source retrieves metadata about a workspace. If `id` is not specified, it uses the `workspace_id` from the provider configuration.

-> **Plan:** Available on all Ory Network plans.

## Example Usage

```terraform
# Read the current workspace (from provider config)
data "ory_workspace" "current" {}

output "workspace_name" {
  value = data.ory_workspace.current.name
}

# Read a specific workspace by ID
data "ory_workspace" "other" {
  id = "other-workspace-uuid"
}
```
