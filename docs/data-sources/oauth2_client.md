# ory_oauth2_client (Data Source)

Fetches information about an Ory OAuth2 client.

This data source retrieves the configuration of an existing OAuth2 client by its client ID.

-> **Plan:** Available on all Ory Network plans.

## Example Usage

```terraform
# Look up an OAuth2 client by client ID
data "ory_oauth2_client" "app" {
  id = "existing-client-id"
}

output "client_name" {
  value = data.ory_oauth2_client.app.client_name
}

output "grant_types" {
  value = data.ory_oauth2_client.app.grant_types
}
```
